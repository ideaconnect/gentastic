Under the hood

Built for real GPUs, not just demos

Gentastic runs on stable-diffusion.cpp via GGUF quantized models - small downloads, real performance.

Ready today

NVIDIA CUDA

GTX 10-series through RTX 50-series. Don't have the CUDA Toolkit? Gentastic can fetch a ~570 MB on-demand CUDA runtime for you - no separate install.

Ready today

Vulkan

The stable, fast path on AMD GPUs - including unified-memory APUs like the Ryzen AI Max "Strix Halo" / Radeon 8060S. No ROCm install, no Python.

Coming

AMD ROCm

Vulkan already covers AMD GPUs today. Native ROCm 7 acceleration is being integrated next, as the upstream StableDiffusion.NET ROCm backend matures and gets validated on real AMD hardware.
